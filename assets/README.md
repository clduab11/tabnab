# TabNab Assets

This directory contains visual assets for the TabNab project.

## Required Assets

### Banner Image
- **Filename**: `banner.png` or `banner.svg`
- **Dimensions**: 1200x400 pixels
- **Purpose**: Repository header banner
- **Content**: TabNab branding with tagline
- **Style**: Modern, professional, aligned with brand identity

### Architecture Diagrams
- **Format**: Mermaid (embedded in README) or SVG files
- **Purpose**: Visualize system architecture and data flow
- **Current**: Using Mermaid code blocks in README.md

### Tool Icons (Future)
- **Format**: SVG or PNG (with transparency)
- **Size**: 64x64 pixels recommended
- **Purpose**: Visual representation of each MCP tool
- **Icons needed**:
  - `get_active_tab` - Browser tab icon
  - `navigate_and_extract` - Navigation/document icon
  - `click_element` - Cursor/pointer icon
  - `fill_input` - Text input/form icon
  - `screenshot_tab` - Camera/screenshot icon

### Demo Assets (Future)
- **Format**: GIF (< 5MB) or MP4
- **Purpose**: Demonstrate TabNab in action
- **Content**: Example of AI agent using TabNab to automate browser tasks

## Design Guidelines

### Color Scheme
- Primary colors should align with browser automation theme
- Suggested palette: Blue (trust), Green (success), Orange (action)

### Typography
- Modern, readable sans-serif fonts
- Consider: Inter, Roboto, or system fonts

### Style
- Clean, minimal design
- Professional but approachable
- Tech-focused aesthetic

## Contributing Assets

If you'd like to contribute visual assets:

1. Ensure files are optimized for web (compressed)
2. Use standard formats (PNG, SVG, GIF for images)
3. Include source files if applicable (e.g., Figma, Sketch)
4. Submit via pull request with clear description
